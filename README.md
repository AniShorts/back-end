## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## env
NODE_ENV를 사용하기 때문에   npm install -g win-node-env 으로 초기 설정이 필요하다

또한, .env 폴더에서 두가지 환경변수를 동적으로 사용있다.
```
.env/.development.env
.env/.production.env
```
### .production.env
> 배포용 환경변수
>
>DATABASE_HOST: 데이터 베이스 주소
>
>DATABASE_PORT: 데이터 베이스 포트
>
>DATABASE_USERNAME: 데이터 베이스 유저 이름
>
>DATABASE_PASSWORD: 데이터 베이스 비밀번호
>
>DATABASE_DATABASE: 데이터 베이스 db
>
>SYNCHROIZE: 데이터 베이스 동기화 설정
>
>SERVER_PORT: 서버 포트
---
